# 10Kalkulator 🧮

Proyek **10Kalkulator** adalah sebuah eksperimen dan eksplorasi implementasi program kalkulator dasar menggunakan 10 bahasa pemrograman yang berbeda. Proyek ini bertujuan untuk membandingkan sintaks, struktur, dan pendekatan dari berbagai bahasa pemrograman dalam menyelesaikan logika yang sama.

## 🚀 Bahasa Pemrograman yang Digunakan

Program kalkulator dalam proyek ini diimplementasikan menggunakan 10 bahasa berikut:

1.  **C** - Bahasa pemrograman struktural yang cepat dan efisien.
2.  **C#** - Bahasa berorientasi objek modern dari Microsoft.
3.  **C++** - Pengembangan dari bahasa C dengan dukungan pemrograman berorientasi objek.
4.  **Go** - Bahasa dari Google yang sederhana, cepat, dan efisien.
5.  **Java** - Bahasa berorientasi objek yang populer (*Write Once, Run Anywhere*).
6.  **JavaScript** - Bahasa utama untuk pengembangan dan interaksi web.
7.  **PHP** - Bahasa *scripting* sisi server untuk pengembangan web.
8.  **Python** - Bahasa tingkat tinggi yang mudah dibaca dan sangat serbaguna.
9.  **Rust** - Bahasa sistem yang berfokus pada keamanan memori dan performa tinggi.
10. **TypeScript** - *Superset* dari JavaScript yang menambahkan sistem tipe statis (static typing).

## 📁 Struktur Direktori

Setiap implementasi bahasa pemrograman memiliki direktorinya masing-masing.

```text

├── Python/
├── Rust/
└── TypeScript/
```

## 🛠️ Cara Menjalankan

Karena setiap program ditulis dalam bahasa yang berbeda, cara menjalankan (*compile/run*) programnya pun berbeda-beda. Silakan masuk ke masing-masing direktori untuk menjalankan kode sesuai dengan *environment* atau *compiler* bahasa tersebut.

Beberapa contoh cara menjalankannya:
- **Python**: `python main.py`
- **Go**: `go run main.go`
- **Rust**: `cargo run` (atau compile dengan `rustc`)
- **JavaScript**: `node main.js`
- **TypeScript**: Compile ke JS dengan `tsc main.ts` lalu jalankan dengan `node main.js` (atau langsung dengan `ts-node`)
- **PHP**: `php index.php`
